## Weather Application (Python)

A Python application that retrieves real-time weather data using the OpenWeatherMap API,
stores results in a SQLite database, and displays stored data using a Tkinter GUI.

### Features
- Fetch live weather data by city name
- Prevents duplicate city entries
- Stores data locally using SQLite
- Scrollable GUI display
- Environment variables for API key security

### Technologies Used
- Python
- Requests
- SQLite
- Tkinter
- dotenv
